== Description ==

Плагин «The post author info» выводит информацию об авторе записи из профиля в панели управления WordPress.

P.S: Если у вас остались какие-либо вопросы пишите на почту <cite>info[@]artemsannikov.ru</cite>.

== Installation ==
	
1. Загрузите плагин "The post author info" в директорию `/wp-content/plugins/`;
2. Активируйте "The post author info" на странице `Плагины` в панели управления.
3. Добавьте следующий код: <?php tpai_author_info(); ?> в нужное место шаблона.
